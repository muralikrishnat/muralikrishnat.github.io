Autocomplete Listbox
Floating command palette that appears above the input box as the user types. Anchored to the input, opens upward. Groups commands by source: recent, bookmarked, history, built-in. Keyboard-navigable with ↑ ↓ to move, Tab/Enter to confirm, Esc to dismiss.
