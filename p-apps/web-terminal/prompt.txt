Build web page for web terminal with below features:
- Ability to Add new tabs.
- Show list of folders and command list under each folder.
- The command input box will stick to bottom always.
- List of command history will have some information like status (tag), datetime, no of times executed, time took to execute, can be shown on hover or kind of expand.
- Ability to re-run the command, delete command.
- Each command set is input and output.
- Command input box will show information like current working directoy, git branch.
- Command input box will also show the recommened commands.
- Ability to set sequential command execution.
- Ability to run parallel command execution, merge output into one with end of the line shows which command.
- Ability to split tabs both horizontal and vertical.
=====instructions for app======
Use glassmorphic.
It should be simple, UI rich looking, with suitable icons, easy to operate.
Use only html, css, javascript and tailwindcss
